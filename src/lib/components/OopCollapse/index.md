Components | Data Entry | OopCollapse  | 折叠栏面板横向显示 |

我们为 collapse 增加了一种排列方式：

-  横向排列：面板水平排列；

## 何时使用

需要折叠栏横向显示时。

## API
#### OopCollapse

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| panelList |  面板列表 | array | - |
| horizontal | 面板显示方向 `horizontal`（横向） | boolean | false |

其他属性参见[collapse](https://ant.design/components/collapse-cn)
