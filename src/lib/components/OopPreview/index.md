## 何时使用

当要预览一个图片的时候使用

## API
#### OopPreview

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否显示 | boolean | false |
| onCancel | 点击取消预览的时候触发 | function | - |
| img | 要预览的图片 | Object{src: 图片地址, alt: 失败的占位} | - |
